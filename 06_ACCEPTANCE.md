# Acceptance Criteria

## Widget
- Loads on any website
- Opens chat UI
- Sends messages successfully

## Admin Panel
- Displays incoming chats
- Sends replies
- Shows message history

## Multi-Tenant
- Different siteIds do not see each other
- Admin sees only own chats

## Performance
- Message delivery < 1s
- No console errors

## Demo
- Widget connected to test website
- Live chat demo recorded
