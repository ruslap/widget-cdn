# Task: Admin Panel (Vercel)

## Goal
Provide an interface for operators to manage chats.

## Stack
- Next.js
- Tailwind CSS
- shadcn/ui
- socket.io-client

## Pages
- /login
- /chats

## Features
- Login
- View list of chats
- Open chat
- Send messages
- Receive messages in real time

## UX requirements
- Clean modern UI
- Chat list + active chat view
- Auto-scroll to latest message

## Deliverables
- Deployed Admin Panel on Vercel
- Connected to API server
